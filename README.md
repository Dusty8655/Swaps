# Eco-Nest Living (PWA prototype)

A lightweight React + Vite single-page experience that maps the Eco-Nest Living product spec into a mobile-first UI with bottom tab navigation.

## Features

- Home, Calculators, MyPlan, NewTech, and Profile screens
- Featured upgrades + new tech cards
- Premium upsell panel and calculator KPI snapshot
- Mobile-friendly bottom tab navigation

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:5173` to view the app.
